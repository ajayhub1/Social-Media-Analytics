# Social Media Analytics Database 📊

A comprehensive MySQL database project designed to simulate a social media platform's backend analytics. This database tracks and measures user behavior, engagement metrics, ad campaigns, and messaging traffic.

## 🚀 Key Features & Schema Design
The project consists of 20+ interconnected tables mapped to handle standard social media functionalities:
* **User Management:** `Users`, `User_Profiles`, `User_Sessions`, `Followers`.
* **Content & Engagement:** `Posts`, `Post_Media`, `Comments`, `Replies`, `Likes`, `Shares`.
* **Classification:** `Hashtags`, `Post_Hashtags`, `Categories`, `Post_Categories`.
* **Marketing & System:** `Ad_Campaigns`, `Ad_Creatives`, `Ad_Performance`, `Notifications`, `Conversations`, `Messages`.
* **Analytics Engine:** `Post_Analytics`, `User_Analytics`, `Engagement_Metrics`, `Daily_Activity`, `Traffic_Sources`.

---

## 🛠️ Tech Stack
* **Database:** MySQL / MariaDB
* **Concepts Used:** Primary & Foreign Keys, Indexes, Auto-Increment, Joins, Group By, Subqueries, and Views.
